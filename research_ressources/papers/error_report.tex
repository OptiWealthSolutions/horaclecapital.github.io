\section*{Analyse Préliminaire}
\begin{itemize}
    \item \textbf{Erreur Interface (GUI) :} Le composant Tkinter lève une erreur \texttt{TclError} car il essaie de mettre à jour une barre de progression avec une chaîne de caractères (\texttt{"Fetching SP500 tickers..."}) au lieu d'une valeur à virgule flottante attendue.
    \item \textbf{Erreur Réseau (Data Engine) :} La fonction \texttt{pd.read\_html} essaie de récupérer des données depuis une URL (vraisemblablement Wikipédia pour le S\&P 500). L'accès est bloqué avec un code \textbf{403 Forbidden}, ce qui indique que le serveur rejette la requête. Cela est généralement dû à l'absence de headers \texttt{User-Agent} appropriés dans la requête de \texttt{pandas}.
\end{itemize}
