# Post LinkedIn - Horacle Academy (Carrousel)

**Texte du post :**

> L'overdose d'information tue votre performance sur les marchés.
>
> La majorité des investisseurs passent leur temps à réagir au dernier chiffre de l'inflation ou à la dernière news sensationnaliste, sans comprendre les forces de fond qui déplacent réellement les capitaux.
>
> Pourquoi ? Parce qu'ils confondent le bruit et le signal.
>
> La clarté est aujourd'hui le seul véritable avantage compétitif. Et la clarté ne s'obtient qu'avec une méthode.
>
> Avec **Horacle Academy**, nous lançons un mentorat stratégique pour ceux qui veulent construire leur propre framework de décision, basé sur les standards institutionnels.
>
> Ce que nous couvrons :
> 🔹 **Analyse de Flux :** Maîtrise des cycles de liquidité mondiaux.
> 🔹 **Framework de Recherche :** Transformer la donnée brute en thèses d'investissement solides.
> 🔹 **Clarté Décisionnelle :** Agir quand les probabilités fondamentales convergent.
>
> Ce n'est pas une formation théorique déconnectée du réel. C'est un transfert de compétences appliqué aux conditions actuelles du marché.
>
> L'objectif n'est pas de vous donner notre avis, mais de vous apprendre à forger le vôtre pour le reste de votre carrière.
>
> 📩 **Le mentorat est sélectif. Postulez pour le prochain batch ici :** [Insérer Lien Calendly / Page Academy]
>

**Instructions d'export du PDF (Rappel) :**
1. Lancer le serveur local : `npm run dev`
2. Aller sur : `http://localhost:4321/tools/linkedin-carousel`
3. Cliquer sur le bouton bleu "Exporter en PDF"
4. Paramètres : Marges = Aucune, Graphiques d'arrière-plan = Cochés
*(Ou utiliser le script de génération Puppeteer : `node scripts/generate_carousel_pdf.js`)*
