## Comment publier un rapport

1. Dupliquer ce fichier dans `src/content/rapports/`
2. Nommer le fichier `rapport-AAAA-MM-JJ.md` (slug dans l'URL)
3. Remplir le frontmatter (titre, description, date…)
4. Écrire le contenu en Markdown ci-dessous
5. `git commit` + `git push` → GitHub Actions build automatiquement
