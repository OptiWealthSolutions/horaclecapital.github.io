## Comment publier une recherche

1. Dupliquer ce fichier dans `src/content/recherches/`
2. Nommer le fichier `recherche-AAAA-MM-JJ.md` (slug dans l'URL)
3. Remplir le frontmatter (titre, description, date, type…)
4. Écrire le contenu en Markdown ci-dessous
5. `git commit` + `git push` → GitHub Actions build automatiquement
